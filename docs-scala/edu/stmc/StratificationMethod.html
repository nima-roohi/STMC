  16.  def getProgress(iters: Int, sampler: Sampler): Int

    Get an indication of progress so far for simulation, i.e.

    Get an indication of progress so far for simulation, i.e. an approximate value for the percentage of work (samples) done. The value is a multiple of 10 in the range [0,100]. This estimate may not be linear (e.g. for CI/ACI where 'iterations' is computed). It is assumed that this method is called *after* the call to isCompleted(...). Note: The iteration count may exceed that dictated by this method, e.g. if multiple properties are being simulated simultaneously. TODO: check methods for this

    iters

    The number of iterations (samples) done so far

    sampler

    The Sampler object for this simulation
