HypTest 

abstract class HypTest extends SimulationMethod

Same as SimulationMethod, with a few default implementations and a few more functions that help testing the class without using Sampler or Expression.

Note

Almost every method in SimulationMethod is declared here as well (even if it is not given a default implementation). So ScalaDoc will inherit the documentation in children of this class.
