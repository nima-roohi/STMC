  1.  new GSPRT(threshold: Double, alpha: Double, beta: Double, minSamples: Int)

    Construct a hypothesis test in which the null hypothesis is p < t and the alternative hypothesis is p > t, where p is the actual probability and t is the input threshold.

    Construct a hypothesis test in which the null hypothesis is p < t and the alternative hypothesis is p > t, where p is the actual probability and t is the input threshold.

    threshold

    Input Threshold

    alpha

    Type I Error Probability (also known as 'false positive' probability).

    beta

    Type II Error Probability (also known as 'false negative' probability).

    minSamples

    Minimum number of samples before making any decision 

    The following requirements must be met by the main constructor:
  0 < threshold < 1
  0 < alpha < 0.5
  0 < beta < 0.5
  1 < minSamples
